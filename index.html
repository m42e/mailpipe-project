Goals

Create a stand alone mailserver to receive emails (port 25) and a webserver Get notifications pushed to a callback url when an email matching your route is received. Enable the creation and deletion of routes via an api.

Installation

$ git clone git://github.com/readevalprint/mailpipe-project.git

MailPipe

An opensource email to REST api project

Download this project as a .zip file Download this project as a tar.gz file

Goal

Get notifications pushed to a callback url when an email matching your route is received. Provide an API to manage routes, callbacks, and emails.

Installation

$ git clone git://github.com/readevalprint/mailpipe-project.git
cd ./mailpipe-project
pip install -r requirements.txt

Usage

  1. Create a user
  2. Log in as that user
  3. Create a route and callback url
  4. Email that route (+(*.))?@YOUR_HOST
  5. Handle the callback on your server and request the corresponding email id

MailPipe

An opensource email to REST api project

View the Project on GitHub readevalprint/mailpipe-project

  • Download ZIP File
  • Download TAR Ball
  • View On GitHub

Goals

Create a stand alone mailserver to receive emails (port 25) and a webserver Get notifications pushed to a callback url when an email matching your route is received. Enable the creation and deletion of routes via an api.

Installation

$ git clone git://github.com/readevalprint/mailpipe-project.git
cd ./mailpipe-project
pip install -r requirements.txt

Usage

  1. Create a user
  2. Log in as that user
  3. Create a route and callback url
  4. Email that route(+(*.))?@YOUR_HOST
  5. Handle the callback on your server and request the corresponding email id
